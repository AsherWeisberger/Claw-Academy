# Claw Academy - Analytics & Tracking

## Where to Track

### GitHub Pages Analytics
- Go to repo Settings > Pages
- Enable GitHub Pages
- Check "Enforce HTTPS"

### External Tools (Free)
- **Google Analytics** - Add tracking ID to index.html
- **Plausible** - Privacy-friendly analytics (plausible.io)
- **Fathom** - Simple analytics (usefathom.com)
