# Claw Academy - Business Setup Guide

## Gumroad Setup (DO THIS FIRST)

### 1. Create Account
- Go to gumroad.com
- Sign up with Google or email

### 2. Create Product - Course ($47)
1. Click "New Product"
2. Name: "Claw Academy Course"
3. Price: $47
4. Description: Copy from below

**Description:**
```
Learn OpenClaw in 6 lessons. Build your own AI assistant.

WHAT'S INCLUDED:
- 6 step-by-step lessons
- Installation to advanced automation
- Access to community Discord
- Lifetime updates
- 30-day money-back guarantee

LESSONS:
1. What is OpenClaw?
2. Installation & Setup  
3. Connect Your First App
4. Voice & Audio
5. Automation Basics
6. Custom Agents
```

5. Upload course files (PDFs or video links)
6. Click "Publish"

### 3. Create Product - 1-on-1 ($97)
1. Click "New Product"
2. Name: "Claw Academy - 1-on-1 Setup Call"
3. Price: $97
4. Description: "60-minute personal Zoom call to help you set up OpenClaw"
5. Click "Publish"

### 4. Create Product - Mastermind ($97/mo)
1. Click "New Product"
2. Name: "Claw Academy Mastermind"
3. Price: $97/month (recurring)
4. Description: "Monthly Zoom calls, Slack access, direct support"
5. Click "Publish"

### 5. Get Links
- Course: https://gumroad.com/l/YOUR-COURSE-LINK
- 1-on-1: https://gumroad.com/l/YOUR-SETUP-LINK
- Mastermind: https://gumroad.com/l/YOUR-MASTERMIND-LINK

### 6. Update Website
Replace Gumroad links in index.html with your actual links!

## Zoom Setup

### 1. Create Free Account
- Go to zoom.us
- Sign up with Google or email

### 2. Create Recurring Meeting
1. Click "Host a Meeting"
2. Set recurrence: "Weekly"
3. Day: Saturday
4. Time: 2:00 PM ET
5. Duration: 60 minutes

### 3. Get Meeting Link
- Copy the join link
- It will look like: https://zoom.us/j/123456789

### 4. Add to Website
- Replace "Zoom link" in marketing with your actual link

## Email Service - ConvertKit (Free up to 300)

### 1. Sign Up
- Go to convertkit.com
- Sign up free

### 2. Create Form
1. Click "Forms" > "Create Form"
2. Choose "Inline" 
3. Customize to match your site
4. Get embed code

### 3. Create Sequence
1. Click "Automations" > "Sequences"
2. Create new sequence
3. Add emails from email-sequence.md
4. Set triggers

### 4. Connect Form
- When someone subscribes, add to "Welcome Sequence"

## Discord Server Setup

### 1. Create Server
1. Open Discord
2. Click + > "Create My Own"
3. Name: "Claw Academy"

### 2. Create Channels
```
📚 COURSE
  ├── #start-here
  ├── #lesson-1
  ├── #lesson-2
  ├── #lesson-3
  ├── #lesson-4
  ├── #lesson-5
  └── #lesson-6

💬 COMMUNITY
  ├── #introductions
  ├── #general
  ├── #help
  └── #showcase

📢 ANNOUNCEMENTS
  └── #news
```

### 3. Set Up Roles
- Student (default)
- Graduate (completed course)

### 4. Get Invite Link
1. Server Settings > Invite Links
2. Create new link
3. Set to never expire (or long duration)
4. Copy link for website
